# Video Checklist

Use this for the hackathon demo recording.

## Before Recording

- OpenClio is running locally
- demo task is ready
- terminal is readable
- secret env values are hidden
- trust/approval settings are visible if relevant

## Demo Sequence

1. Briefly state the problem:
   - trusted local agent execution for real work
2. Show the project/branch and the OpenClio Sentinel framing
3. Run the demo through `demo_runner.sh`
4. Show:
   - user task
   - agent response
   - exported evidence bundle
   - trust-policy summary
5. Close with:
   - why this fits `Agents that trust`
   - what OpenClio contributes as the real runtime

## Capture Checklist

- terminal output is legible
- no API keys visible
- final evidence files shown on screen
- short explanation of why the user can trust the system
